## 📁 Featured Projects

### 🔵 [splunk-home-soc-lab](https://github.com/hoqueariful/splunk-home-soc-lab)
Home SOC built on VMware running Splunk Enterprise. Ingests Windows Event Logs (Event ID 4688) and firewall logs, uses SPL to detect repeated blocked outbound connections and privilege escalation patterns. Includes a working dashboard and documented detection scenarios.
`Splunk` `SIEM` `Blue Team` `Incident Response` `PowerShell`

### 🔵 [CyberSentinel](https://github.com/hoqueariful/CyberSentinel)
Python security automation suite: log parser (detects SQLi, XSS, brute-force, path traversal), multi-threaded port scanner, and web vulnerability checker (SSL/TLS, CORS, security headers, risk scoring 0–100). 1,200+ lines of production-ready code.
`Python` `SOC Automation` `Vulnerability Assessment` `Log Analysis`

### 🔵 [Phishing-Email-Analysis-Lab](https://github.com/hoqueariful/Phishing-Email-Analysis-Lab)
Structured analysis of real-world phishing campaigns: full email header inspection, URL and domain IOC extraction, WHOIS lookups, and written investigation reports. Covers BEC, credential harvesting, and impersonation techniques.
`Threat Analysis` `Phishing` `DFIR` `IOC` `Blue Team`

### 🔵 [serverpeek](https://github.com/hoqueariful/serverpeek)
Bash-based web server fingerprinting tool for security reconnaissance: DNS, WHOIS/ASN, HTTP headers, CDN/WAF detection, SSL certificate analysis, passive subdomain enumeration, and port scanning — all in one automated report.
`Bash` `Recon` `Bug Bounty` `Network Security`

### 🔵 [TeleTalk-Customer-Analysis](https://github.com/hoqueariful/TeleTalk-Customer-Analysis)
Python/Pandas data analysis of 300 simulated telecom customers — churn modelling, behavioural segmentation, and visualisation. Demonstrates data-driven security analytics capability.
`Python` `Pandas` `Data Analysis`
